# LLM Observability with LangSmith

This repository demonstrates LLM observability, tracing, monitoring, and evaluation using LangSmith along with LangChain and LangGraph.

The project includes:
- Simple LLM calls
- Sequential chains
- RAG pipelines
- AI agents
- LangGraph workflows
- Tracing & monitoring with LangSmith

---

## 🚀 Features

- LLM tracing and observability
- Sequential chain implementation
- Retrieval-Augmented Generation (RAG)
- AI Agent workflow
- LangGraph integration
- Prompt execution monitoring
- Evaluation and debugging of LLM applications
